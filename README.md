# Alien_Dungeons
one of my one python games rewritten in C++ with SDL

![screenshot](https://github.com/Ztirom45/Alien_Dungeons/blob/main/screenshots/screenshot2.png)

##### Features

- room system
- animation system
- path finder

##### requred packages(Linux)
- SDL2_image
- SDL2_image-devel
- SDL2_ttf
- SDL2_ttf-devel
- freeglut-devel
- glm-devel

##### Building (Linux)


- compile: `$ make`
- run: `$ ./bin/main`
